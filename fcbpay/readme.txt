FCB payment plugin for WooCommerce 

== Description ==

FCB payment plugin is aim to provide cooperative special stores and individual members who are using Woo commerce that requires payment processing without having to deal with complicated upfront settings on their own.
You can directly install and set the plugin to with FCB pay.

= All in one payment solution =
- Credit card (one-off payment, instalment payment, regular fixed amount), ATM, Web ATM, codes/bar codes, you can use it without any registration.
- There is no need to apply for credit card service with individual banks, as long as you register for First Bank, you can enjoy a variety of payment methods.
- Single or multiple cash flows can be set to be displayed to consumers according to their needs.

== Installation ==

= system Requirements =

- PHP version 5.6.0 or greater
- MySQL version 5.5 or greater

= Automatically Installation =
1. Log in to your WordPress dashboard, visit the "Plugins Menu" and click "Add".
2. Enter "FCBPay Payment for WooCommerce" in the "search field" and click search.
3. Click "Install" to start the installation.
